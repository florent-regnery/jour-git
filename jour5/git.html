Commande Git Description
git clone Récupère un fichier distant
git add . Ajouter un nouveau contenu au commit
git commit-m Sert à commanter son commit et à le valider
git push Envoyer ses fichiers sur le repository
git pull Récupère la dernière version de la branche
git fetch Récupère toute les infos de la muse à jour
git merge Fusionner deux branches
git checkout-b Créer une nouvelle branche ou switch vers une autre branche déjà crée
